# 2026-03-24 玩家属性与测灵流程落地记录

## 本轮目标

暂停前端预留分支，转回核心玩法建模。

本轮聚焦两件事：

- 建立统一玩家角色模型
- 把“测灵石确认灵根”作为正式玩法节点接入主线

## 已落地

### 1. 玩家角色模型统一收口

- 新增 `systems/character_model.py`
- 当前角色已具备统一结构块：
  - `identity`
  - `progression`
  - `primary_stats`
  - `combat_stats`
  - `currencies`
  - `equipment`
  - `affinities`
  - `reserves`

### 2. 新角色改为凡人起步

- 角色默认阶段改为 `mortal`
- 默认境界文案改为 `凡人`
- 初始状态不再默认进入 `炼气一层`

### 3. 五行单灵根进入模型层

- 当前支持：
  - 金
  - 木
  - 水
  - 火
  - 土
- 灵根定位当前只落到：
  - 战斗倾向
  - 技能方向标签
  - 生活亲和标签
- 本轮不实现正式技能

### 4. 水灵根额外补入液体亲和

- 水灵根当前生活亲和为：
  - 符箓
  - 液体

### 5. 货币模型升级

- 凡人阶段主货币为 `铜钱`
- 已确认灵根后主货币切为 `灵石`
- `铜钱` 保留为辅助货币

### 6. 测灵石流程接入主线

- 新增主线阶段：
  - `测灵定根`
- 第三段主线完成后，不再直接入道
- 玩家需要前往 `升仙台`
- 使用：
  - `触发 测灵石`
  - `触发 测灵石 金|木|水|火|土`
- 确认后才切到 `cultivator`

### 7. 新手村与外门之间新增过渡场景

- 新增场景：
  - `升仙台`
- 场景定位：
  - 山下试炼完成者集中候测的过渡区域
  - 大批量候选者等待测灵的地方
  - `测灵石` 与相关维序 / 引导 NPC 摆放在这里
- 当前路径更新为：
  - 新手村
  - 升仙台
  - 青云外门前坪

### 8. 当前明确未做

- 不立刻入宗
- 不实现正式技能
- 不重做商店 / 坊市 / 交易的双货币结算
- 不继续扩前端页面

## 2026-03-25 补充

### 9. 启灵过渡阶段补齐

- 测灵后不再直接视为 `炼气一层`
- 当前流程改为：
  - 测灵后进入 `启灵`
  - 与 `引灵执事` 交谈完成 `引气入体`
  - 再进入 `炼气一层`

### 10. 主线补出引气入体节点

- 新增主线阶段：
  - `qi_guidance_ready`
- `引灵执事` 现在承担过渡引导
- 玩家完成该步骤后，当前入门试炼主线才真正 `completed`

### 11. 代码模块化收口

- 新增 `systems/object_index.py`
  - 统一按 `content_id` 查找 live 对象
- `systems/quests.py`
  - 新增状态 helper 与管理员测试前置态准备 helper
- `commands/devtools.py`
  - 不再手动散写多组主线 flag
- `world/start_area.py`
  - 改为优先按 `content_id` 更新已有对象 / NPC
  - 解决 JSON 已更新但运行态对象属性未同步的问题
