# 玩家角色属性建模 V1

更新日期: 2026-03-25

## 目标

本轮目标不是继续扩前端，而是把“玩家角色”从零散状态字段收口成后续可持续扩展的统一模型。

本版重点解决：

- 玩家从凡人起步，而不是默认炼气一层
- 灵根作为真正的职业原型进入角色模型
- 货币从单一铜钱升级为“阶段化主货币”
- 后续装备、功法、生活专精、灵宠有稳定挂载点

本版不做：

- 正式技能实现
- 完整战斗公式和平衡
- 宗门正式入门流程
- 生活系统产出链
- 灵宠玩法

## 当前结论

### 角色阶段

角色分为两个阶段：

- `mortal`
  - 凡人未入道
  - 无灵根
  - 无灵力体系
  - 只有最小生存与新手玩法属性
- `cultivator`
  - 已确认主灵根
  - 已进入修行体系
  - 开始拥有灵力、灵根倾向和后续术法扩展入口

### 成长主轴

- 本项目当前仍以 `境界 / 修为` 作为主成长轴
- 不单独引入传统 RPG 的“等级”
- 凡人阶段可显示为 `凡人`
- 修士阶段再进入 `炼气` 体系

### 一级属性

统一采用五维底盘：

- `体魄`
- `灵元`
- `神识`
- `身法`
- `根骨`

设计原则：

- 一级属性是统一底盘
- 灵根负责提供成长倾向和职业方向
- 装备、buff、功法、灵宠后续都挂在统一派生链上

### 战斗派生属性

当前已在模型层预留：

- `hp / max_hp`
- `mp / max_mp`
- `stamina / max_stamina`
- `attack_power`
- `spell_power`
- `defense`
- `speed`
- `crit_rate`
- `crit_damage`
- `healing_power`
- `shield_power`
- `control_power`
- `control_resist`
- `threat_modifier`

本轮只建立字段和最小派生规则，不做正式乘区和平衡设计。

## 五行灵根定位

### 金灵根

- 定位：单体伤害、杀伐、较高防御
- 强项：
  - 单体输出高
  - 伤害权重高
  - 防御仅次于土
- 弱项：
  - 灵力消耗高
  - 冷却偏长
  - 速度偏慢
- 生活亲和：
  - 灵器
  - 金属

### 木灵根

- 定位：高速辅助控制、持续伤害、群体恢复倾向
- 强项：
  - 血量高
  - 速度最高
  - 控制与持续伤害能力强
- 弱项：
  - 单点爆发低
  - 伤害需要时间累积
- 生活亲和：
  - 炼丹
  - 采集
  - 草木

### 水灵根

- 定位：单体治疗、净化、护盾、抗控
- 强项：
  - 单体恢复最强
  - 护盾最强
  - 抗控强
  - 速度仅次于木
- 弱项：
  - 输出最低
  - 单人刷怪效率弱
- 生活亲和：
  - 符箓
  - 液体

### 火灵根

- 定位：爆发、范围伤害、暴击、灼烧
- 强项：
  - 灵力高
  - 暴击率最高
  - 范围与爆发强
- 弱项：
  - 技能冷却最长
  - 稳定性一般
- 生活亲和：
  - 炼器
  - 火焰

### 土灵根

- 定位：承伤、防御、嘲讽、反击、反伤
- 强项：
  - 防御最高
  - 威胁值最高
  - 承伤稳定
- 弱项：
  - 速度慢
  - 输出低
  - 消耗和冷却偏长
- 生活亲和：
  - 阵法
  - 大地

## 货币模型

货币不再只保留单一 `copper` 语义，而是纳入统一角色模型。

### 凡人阶段

- 主货币：`铜钱`
- 当前凡俗补给、基础商店、训练期消耗仍可继续使用铜钱

### 已入道阶段

- 主货币：`灵石`
- 辅助货币：`铜钱`

当前行为原则：

- 凡人阶段 `primary_currency = copper`
- 已确认灵根后 `primary_currency = spirit_stone`
- 铜钱不清空，保留为凡俗交易和低阶杂项货币
- 现有商店 / 坊市 / 交易逻辑暂时仍主要按铜钱兼容，后续再逐步升级

## 装备与扩展预留

本轮只开放两个装备槽：

- `chest`
- `legs`

当前定位：

- 通用防具
- 不限制灵根佩戴
- 后续通过灵根倾向标签体现契合度

装备结构当前预留：

- 槽位
- 基础属性
- 一级属性加成
- 战斗属性加成
- 灵根倾向标签
- 词条扩展位

同时预留但不实现：

- 其他装备槽位
- 功法与术法系统
- 灵宠数据位
- 生活专精等级
- 多灵根 / 异灵根

## 测灵流程

### 任务节点

当前新手主线不再在第三段战斗后直接结束，而是改成：

1. `渡口试手`
2. `石阶试锋`
3. `溪谷巡查`
4. `测灵定根`
5. `引气入体`

其中第四段的任务目标是：

- 前往 `升仙台`
- 触发 `测灵石`
- 确认自己的灵根

其中第五段的任务目标是：

- 留在 `升仙台`
- `交谈 引灵执事`
- 完成第一次引气入体

### 场景定位

`升仙台` 是新手村与正式外门之间的过渡场景。

场景定位为：

- 大量候选者集中等待测灵
- `测灵石` 与相关维序 / 引导 NPC 驻守
- 这里只决定“灵根确认”，不等于已经正式入宗

当前推荐路径：

- 新手村
- 升仙台
- 青云外门前坪

### 交互入口

测灵流程不走 NPC 直接选项，而走对象触发：

- `触发 测灵石`
  - 返回测灵说明和五行选项
- `触发 测灵石 金`
- `触发 测灵石 木`
- `触发 测灵石 水`
- `触发 测灵石 火`
- `触发 测灵石 土`
  - 用于正式确认灵根

### 当前规则

- 只有当主线处于 `root_choice_ready` 或后续引气阶段时，测灵石才会回应
- 若角色已完成灵根确认，再次触发只返回“已定灵根”的说明
- 确认灵根后：
  - 角色阶段切换为 `cultivator`
  - 主灵根写入角色模型
  - 主货币切换为 `灵石`
  - 主线推进为 `qi_guidance_ready`
  - 境界进入 `启灵`
- 完成 `引气入体` 后：
  - 境界推进为 `炼气一层`
  - 主线推进为 `completed`
- 本轮只确认灵根，不立刻入宗
